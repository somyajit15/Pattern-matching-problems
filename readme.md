# Pyramid Star Pattern

## Problem Statement

Given an integer `N`, print a centered pyramid pattern consisting of `N` rows using the `*` (asterisk) character.

For each row:

* The number of leading spaces decreases as the row number increases.
* The number of stars follows the pattern `2 × row - 1`.
* The pyramid should be perfectly centered.

### Example

**Input**

```text
5
```

**Output**

```text
    *
   ***
  *****
 *******
*********
```

### Constraints

* 1 ≤ N ≤ 100
